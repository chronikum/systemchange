# SYSTEMCHANGE

## A MONITORING TOOL FOR CACHET, WRITTEN IN TYPESCRIPT

### Requirements
 - Cachet (of course)
 - A Cachet-API key (find it in your user profile)
 - Node

### Installation

- Create a `.env` with following keys:

    ```
    TOKEN=exampletoken
    BASEURL=https://example.com
    ```

- Run `npm i` to setup dependencies.
- To start: `npm start`
